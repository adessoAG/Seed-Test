HOW TO START

To log in to Seed-Test you either need a Seed-Test account or you can log in with your Github account. To create a Seed-Test account, click on "Register". If you already have a GitHub account, you can connect directly via the GitHub Login and access your GitHub projects directly as repositories.

As soon as you logged in successfully you can select the repository/project you want to use from the appearing list.

Via the account settings you can create new projects (repositories) or connect your account with your GitHub account or connect to your Jira server to load existing projects. Alternatively, you can also create a custom repository, for what you do not need a Github or Jira account.

The settings and connections can be edited using the account settings via the menu. You can also change the repository here.

In our tutorial we explain in detail how to create stories, perform tests and how to use the other features in our application.
