How to start

Welcome to seed-test!

If you don't have an account, you have two possibilities to change this:


1. No requirements

Register and log in with your created account data via the login on the right. You will be forwarded directly to the account settings where you can create new projects (repositories) or connect your account to Github or Jira to load existing projects. You can edit the settings and connections at any time using the menu > Account Settings.



2. Github account required

If you already have a Github account, you can connect directly via the Github login. With this option you can access your github projects directly as repositories. You can change the settings and connections at any time using the menu > Account Settings. Additionally you can connect to Jira or create new projects (repositories) directly using the account settings.



If you have any questions, feel free to contact us at any time: seed-test@adesso.de
